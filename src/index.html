Чеклист:

  • Переместить из модулуса несколько индикаторов готово
  • Завернуть в реакт вторник
  • Переехать на тайпскрипт среда
  • Пофиксить порядок вывода готово
  • Сделать элементы редактируемыми (перетаскиаемыми и удаляемыми) среда
  • Добавить режимы вывода (bars, colored bars, renko и т.д.) среда
  • Добавить настройки для инструментов и индикаторов (для одного для начала) четверг
  • Добавить элементы (вектор, отрезок, горизонтальная линия, свободная линия, квадрат, круг, треугольник, спираль фибоначи, круг фибоначи, текст) и ластик четверг
  • Переместить индикаторы из модулуса (ещё нужно скоупнуть) пятница или суббота
  • Добавить настройки для чарта неизвестно
